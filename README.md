# VerdeAI Tester Beta v1.26

Goal-aware public tester build.

## New in v1.26

- Added goal selection before upload.
- Added optional postcode field.
- Added goal-aware Property Story, Property DNA, Opportunity Meter, timeline and futures.
- Added two new futures: Living Nature Edge, Useful Food Pocket, Street Appeal Anchor.
- Improved accessibility with larger touch targets, keyboard-selectable future cards and status text.
- Improved reduced-motion support.
- Refactored the JavaScript into goal/profile driven rendering instead of fully fixed text.
- Updated version numbers to v1.26.
- Added CHANGELOG.md.

## Deploy

1. Extract this ZIP.
2. Copy the contents into your local `VerdeAI-Alpha` folder.
3. In GitHub Desktop, use summary: `Build VerdeAI Tester Beta v1.26`
4. Commit to `main`.
5. Push origin.
6. Cloudflare Pages will deploy automatically.
