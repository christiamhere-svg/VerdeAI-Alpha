# VerdeAI v6.5 Workshop Build

VerdeAI v6.5 keeps the public tester experience and adds clearer planning for the first real AI render provider.

Real rendering is not connected yet. GitHub Pages can host the public interface, but API keys must live in a backend/proxy such as Netlify Functions, Cloudflare Workers, Vercel, or a small backend.

Use the app as a static public beta until a backend provider is chosen and configured.
