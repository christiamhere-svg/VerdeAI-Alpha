## Deploy
1. Extract this ZIP.
2. Copy the contents into your local `VerdeAI-Alpha` folder.
3. GitHub Desktop summary: `Update to Tester Beta v1.25`.
4. Commit to main.
5. Push origin.
6. Cloudflare auto-deploys.
