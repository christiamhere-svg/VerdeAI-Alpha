# VerdeAI v9.0 Workshop Build

VerdeAI is a static public-beta property futures app. Build v9.0 hardens the user-guided concept-placement editor for phone use while keeping real AI rendering disabled.

## Deploy

Upload the complete `VerdeAI_v9_0_Workshop_Build` folder to Cloudflare Pages or deploy the folder contents as the site root.

After deployment:

1. Confirm the header says **Build v9.0**.
2. Analyse a photo or restore an analysed project.
3. Tap **Adjust concept placement**.
4. Confirm the five placement steps, large handles, Undo, Reset and **Done placing concept** appear.
5. Open AI Setup and confirm: rendering disabled, backend not connected, API key not added, paid calls locked, and the one-image pilot is **Prepared · not approved**.

## Important boundary

No live v8.9.2 physical-phone calibration screenshots were attached for this milestone. v9.0 therefore reports Chromium pointer simulation and responsive layout checks separately from physical Android evidence. It does not claim a completed real-device pass.
