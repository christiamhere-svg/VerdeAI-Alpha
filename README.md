# VerdeAI v7.0 Workshop Build

VerdeAI v7.0 adds a practical first-render setup package while keeping real AI rendering disabled by default.

The recommended path is:

1. Keep GitHub Pages as the public frontend for now.
2. Add one small backend/proxy for rendering.
3. Start with Replicate / FLUX.
4. Estimate cost before rendering.
5. Render one future image only.
6. Keep the concept-board fallback.

No paid AI rendering is connected in this build.
