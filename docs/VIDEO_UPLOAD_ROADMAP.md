# Short Walkthrough Video Roadmap — VerdeAI v7.7

Future optional feature. Not active in the current app.

## Recommended path

1. Keep the current photo-first workflow stable.
2. Add optional 5–15 second walkthrough video upload later.
3. Extract key frames on-device or backend-side.
4. Choose the clearest frame as the main property image.
5. Use extra frames for layout clues: access, shade, hidden corners, and scale.

## Why later

Video adds larger files, slower uploads, privacy considerations, device compatibility problems, and possible AI/processing costs. It should stay optional until the photo and render flow are reliable.
