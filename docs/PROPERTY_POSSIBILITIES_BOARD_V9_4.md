# Property Possibilities Board — v9.4

## Product communication goal

A first-time viewer should grasp the complete VerdeAI proposition from one board:

**property today → six futures → recommendation → first move → five-year direction**

## Board anatomy

- **Your Property Today:** untouched photo anchor.
- **Six Possible Futures:** six compact cards using the same photo and the existing future-specific calibrated botanical overlays.
- **VerdeAI Recommendation:** best-fit future based on the current clues.
- **Property Compass:** a compact comparison of belonging, flow, nature, function, ease and possibility.
- **Oracle Reading:** a short interpretation of the property's underlying role.
- **Next Steps:** review, choose, test marker 5, then consider the longer direction.
- **Five-Year Evolution:** the selected future gradually appears over the same photograph as a direction, not a final render.
- **Full-Size View:** preserves the verified calibration editor, Done placing concept, marker 5 and detailed legend.

## Deliberate restraint

The board does not create fake photorealistic renders. It arranges the already verified free overlays into a more convincing explanation. Calibration guides are suppressed only in compact previews; the detailed full-size view remains unchanged.
