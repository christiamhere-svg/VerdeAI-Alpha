# VerdeAI v9.2.2 — Owner-Only First Real Render

Do this only after the Worker health endpoint reports every approved gate ready and Build v9.2.2 is live.

1. Keep all ten tester codes private.
2. Use one code yourself for the owner test.
3. Open VerdeAI in a private browser tab.
4. Confirm the page shows **Public Beta · Build v9.2.2**.
5. Upload a real property photograph that you have permission to use.
6. Complete the normal analysis and five-step concept calibration.
7. Confirm the free calibrated overlay is still usable.
8. Open **AI Setup**.
9. Press **Check live pilot readiness**.
10. Confirm the panel reports the provider path ready and still says no frontend API key.
11. Press **Create one AI concept render**.
12. Verify the property situation, selected future, US$0.15 reservation ceiling, privacy explanation and no-store statement.
13. Enter the single owner test code and complete all four confirmation boxes.
14. Press **Confirm one render** once. Do not retry while a request is still running.
15. Wait up to 120 seconds.
16. Confirm the result is labelled **AI Concept Render · Not Final Design**.
17. Confirm the building, roofline, doors, windows and protected access remain plausible.
18. Confirm the result is early-stage inspiration rather than unrealistic mature landscaping or structural work.
19. Record the Worker request ID and provider request ID if displayed in operational logs; do not record the photograph or prompt in logs.
20. Check the OpenAI usage dashboard and compare the actual charge with the planning assumption.

## Stop conditions

Deploy the safe-lock configuration immediately if any of these occurs:

- More than one provider request starts from one confirmation
- A secret appears in browser code or network responses
- The Worker accepts an invalid or reused invite code
- The spend cap or tester limits do not match the approved settings
- Images or prompts appear in operational logs or storage
- The free calibrated overlay becomes unavailable
- The provider output creates unsafe or seriously misleading structural changes

Do not invite testers until this owner-only first render passes.
